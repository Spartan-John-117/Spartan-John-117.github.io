HOMELAB CYBER

PLATEFORME : Test & Simulation

STATUT : ÉVOLUTIF

Ce homelab sert de socle d'apprentissage continu pour tester des configurations d'infrastructure, valider des mécanismes de défense et reproduire des incidents en environnement contrôlé. L'objectif est de relier l'administration système, le réseau et la cybersécurité avec une approche pragmatique orientée exploitation et détection.

Voir l'architecture Retour projets

[ Architecture Type ]

Composants

  • Hyperviseur local (Proxmox/VirtualBox) avec VLAN de laboratoire.
  • Contrôleur de domaine Active Directory + DNS interne.
  • Serveur Linux durci pour services d'administration (SSH, automation, scripts).
  • Pare-feu/routeur dédié pour filtrage inter-zones et règles NAT.
  • Collecte centralisée des logs via SIEM (Syslog, Windows Events, alerting).
